# CIS-18A-Final-Project
Java Programming Final

MyShoeStore

MyShoeStore is the project I did for this Final, and it is a Java menu-driven business program that allows users to view the shoe catalog,
place an order, schedule a delivery, review their order, and save it to a text file.

Features : 
- View available shoes, sizes, and prices
- Place an order with size and quantity
- Schedule delivery date and time
- View order summary
- Save order information to a text file

Target Audience
This program is designed for beginner Java students and users who want to understand
how a simple business ordering system works in Java.

Techn Used
- Java
- Menu-driven user input/output
